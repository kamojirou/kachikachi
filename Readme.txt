ゲームタイトル「カチカチ」

＜内容＞
「ぷよぷよ」をベースにした作品

＜開発環境＞
　Microsoft Visual Studio Community 2017
　DirectX9.0c

＜動作環境＞
　windows8.1 
  windows10

＜開発期間＞
　6月〜10月(4か月)

＜工夫したこと＞
　初めて制作した「オセロ」との技術の向上を比較してもらうためにサウンドやアニメーションを取り入れました。
サウンドは「オセロ」のときは無音だったので取り入れたことで、よりゲームらしさが出たと思います。
自分好みの環境で遊んでもらいたい思ったので、オプション画面で「背景」と「BGM」を変更できるようにしました。
アニメーションは取り入れたことによってゲームに躍動感を与え、プレイする人に視覚的にも楽しんでもらえるようになったと思います。

まだ関数化ができる箇所はあると思いますが、ほとんどの処理を関数にまとめられたと思うので見やすくなりました。
例えば、ぷよを生成する際、分岐を細かく分けることで様々な組み合わせを具体的にすることができたり、
回転処理は配列をうまく利用し、実際に回転してるように見せたりすることもできました。

＜苦労したこと＞
画像はお借りしたものもありますが、Photoshopを利用して自分で作成したのが一番苦労しました。
その中でも、一枚の画像から切り抜いて描画するのをfor文を利用して書いたのですが、切り抜き元の配置を自分で考えながら作成するのが難しかったです。

また、つながって消す処理やつながっているかどうかを調べる処理はこのゲームの要なので学校の先生に教えてもらいながら、正確に書けたと考えています。
ほかにも、消す処理や浮いているぷよを落とす処理、ゲームオーバーの判定処理などどれから優先的に実行するかを決めるswitch文は大変悩みました。

＜改良したい箇所＞
将来的に二人用で遊べるようにしたいと思っています。
CPUと対戦ができるようにし、遊べる範囲を拡大することでさらに楽しみ方を増やしたいと考えます。

まだ関数にできそうな処理があるのでそれも関数にして読みやすいコードにしたいです。
